워크스페이스

Navigator + Canvas + Studio

Navigator (좌측 패널)

프로젝트의 전체 구조를 관리하는 영역

페이지 관리

페이지 추가, 이름 변경, 삭제

기획 내용 확인

입력한 기획서 내용 다시 보기

전역 규칙 설정

톤앤매너, 조명, 구도, 배경 등 프로젝트 전체 적용

라이브러리

등장인물, 소품 등 에셋을 카테고리별 관리

Canvas (중앙 영역)

Scene과 Cut을 시각적으로 관리하는 메인 작업 공간

Scene-Cut 그리드

  • Scene별로 Cut 카드가 그리드 형태로 배열
  • Cut 카드: 제목, 상태 배지, 썸네일, 스크립트, 메모 수, 버전 수
  • 드래그 앤 드롭으로 순서 변경 (Scene 간 이동 가능)

상단 헤더

  • Scene 수, Cut 수, 승인 수 카운트
  • 프로그레스 바로 전체 진행률 파악
  • 프로젝트 기간 설정

Studio (우측 패널)

Cut 카드를 클릭하면 열리는 상세 작업 공간

Guideline 탭

  • 작업자 배정 — 드롭다운에서 팀원 선택
  • 상태 변경 — 진행중 → 리뷰 요청 → 수정 필요 → 승인
  • 개별 규칙 — 이 Cut에만 적용되는 세부 설정
  • 스크립트 — 해당 Cut의 대사/내레이션
  • 프롬프트 가이드 — AI 이미지 생성용
  • 메모 — 팀원 간 커뮤니케이션

Results 탭

  • AI 모델 선택 — 프리셋 등록 및 선택
  • 콘텐츠 유형 — T2I, Image Editing, T2V 등 5가지
  • 미디어 업로드 — 생성 결과물 업로드
  • 버전 관리 — 자동 버전 생성, 대표 버전 지정, 복원, 삭제
